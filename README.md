# 🏠CozyHouse
Final Project - CozyHouse

---

### 📖프로젝트 소개
현재 운영되고 있는 인테리어 플랫폼인 '오늘의 집'의 스토어 부분을 모티브로 제작한 온라인 인테리어 쇼핑몰 사이트입니다.
상품 카테고리 및 상세페이지, 리뷰 및 문의, 고객센터, 장바구니 및 바로구매, 마이페이지 등 쇼핑몰의 실제 페이지 기능들을 모두 구현하는 것을 목표로 했습니다.

- 개발기간 : 2022.10.11 ~ 2022.11.08
- 인원/역할 : 5명/팀장
